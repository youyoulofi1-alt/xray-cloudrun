# XRAY Cloud Run (VLESS / VMESS / TROJAN)

Deploy Xray-core on Google Cloud Run with WebSocket + TLS.

## Features
- VLESS / VMESS / TROJAN
- Custom UUID / Password
- Custom WS Path
- Optional Custom Domain
- Termux supported
- No Telegram

## Requirements
- Google Cloud account
- gcloud CLI
- Active GCP project

## Usage

```bash
git clone https://github.com/youyoulofi1-alt/xray-cloudrun.git
cd xray-cloudrun
chmod +x install.sh
./install.sh
